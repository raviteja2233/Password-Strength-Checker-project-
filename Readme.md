# 🔐 Password Strength Checker — Tkinter GUI

![Tkinter](https://img.shields.io/badge/Tkinter-GUI-App-green)

A clean and interactive **Password Strength Checker** desktop application built using **Python** and **Tkinter**, with a modern UI and strength progress bar.

---

## 🚀 Features

✔ Simple and intuitive GUI  
✔ Shows password strength level (Weak → Very Strong)  
✔ Progress bar with animated fill  
✔ Provides suggestions to improve password  
✔ Responsive design with show/hide password  
✔ Beginner-friendly and great for portfolio

---

## 💡 How It Works

When a user enters a password:
- It checks:
  - Minimum length  
  - Uppercase letters  
  - Lowercase letters  
  - Numbers  
  - Special characters  
- Based on these checks, it calculates a **score** (0–5)
- Displays:
  - Strength label  
  - Animated progress bar  
  - Suggestions if weak
